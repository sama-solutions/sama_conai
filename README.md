## 🇸🇳 Solution Innovante pour l'Administration Publique Sénégalaise

**SAMA CONAI** est une plateforme révolutionnaire de gestion de l'accès à l'information publique, spécialement conçue pour moderniser l'administration sénégalaise et renforcer la transparence gouvernementale.

### 🎯 Vision

Transformer l'accès à l'information publique au Sénégal en créant un pont numérique transparent, efficace et accessible entre l'administration et les citoyens.

## 🚀 Installation & Déploiement

### ⚡ Installation Rapide

```bash
# Cloner le repository
git clone https://github.com/votre-org/sama_conai.git
cd sama_conai

# Installation des dépendances
pip install -r requirements.txt

# Configuration de la base de données
python manage.py migrate

# Lancement du serveur de développement
python manage.py runserver
```

### 🐳 Déploiement Docker

```bash
# Construction de l'image
docker build -t sama-conai .

# Lancement avec Docker Compose
docker-compose up -d

# Accès à l'application
# http://localhost:8080
```
