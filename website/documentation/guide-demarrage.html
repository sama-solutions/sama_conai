Guide de Démarrage

Installation et première configuration de SAMA CONAI

Sommaire
Prérequis Installation Configuration Premier Démarrage Vérification Prochaines Étapes

Prérequis

Système requis
  • Système d'exploitation : Ubuntu 20.04+ / Debian 10+ / CentOS 8+
  • Python : Version 3.8 ou supérieure
  • Base de données : PostgreSQL 12+ (recommandé)
  • RAM : Minimum 4 GB (8 GB recommandé)
  • Stockage : Minimum 20 GB d'espace libre

Installation

1. Installation d'Odoo

Installez Odoo 16 ou version supérieure :

# Installation via le script officiel
wget -O - https://nightly.odoo.com/odoo.key | apt-key add -
echo "deb http://nightly.odoo.com/16.0/nightly/deb/ ./" >> /etc/apt/sources.list.d/odoo.list
apt-get update && apt-get install odoo
2. Installation du module SAMA CONAI

Clonez le repository et installez le module :

# Cloner le repository
cd /opt/odoo/addons
git clone https://github.com/sama-solutions/sama_conai.git

# Rendre le script d'installation exécutable
chmod +x sama_conai/install.sh

# Exécuter l'installation
./sama_conai/install.sh
Note : Le script d'installation configure automatiquement les dépendances et les permissions nécessaires.

Configuration

Configuration de la base de données

Créez une base de données dédiée :

sudo -u postgres createdb sama_conai
sudo -u postgres createuser --createdb --no-createrole --no-superuser odoo
sudo -u postgres psql -c "ALTER USER odoo WITH PASSWORD 'votre_mot_de_passe';"
Configuration Odoo

Modifiez le fichier de configuration /etc/odoo/odoo.conf :

[options]
addons_path = /usr/lib/python3/dist-packages/odoo/addons,/opt/odoo/addons
db_host = localhost
db_port = 5432
db_user = odoo
db_password = votre_mot_de_passe
dbfilter = sama_conai

Premier Démarrage

1. Démarrer le service Odoo
sudo systemctl start odoo
sudo systemctl enable odoo
2. Accéder à l'interface web

Ouvrez votre navigateur et accédez à : http://votre-serveur:8069

3. Créer la base de données
  1. Cliquez sur "Créer une base de données"
  2. Nom de la base : sama_conai
  3. Email administrateur : votre email
  4. Mot de passe : choisissez un mot de passe sécurisé
  5. Langue : Français
  6. Pays : Sénégal

Vérification de l'installation

Points de contrôle :
  • Service Odoo démarré
  • Base de données créée
  • Module SAMA CONAI visible dans les applications
  • Interface accessible via navigateur
  • Connexion administrateur fonctionnelle
  • Menus CONAI disponibles
