Introduction

SAMA CONAI est un module Odoo conçu pour promouvoir la transparence et la bonne gouvernance au Sénégal.

Fonctionnalités principales :
  • Gestion des demandes d'accès à l'information
  • Système d'alertes citoyennes
  • Tableaux de bord analytiques
  • Gestion multi-utilisateurs
  • Sécurité et confidentialité
  • Interface mobile responsive

Astuce : Utilisez les raccourcis clavier pour naviguer plus rapidement. Appuyez sur Alt + M pour accéder au menu principal.

Gestion des Demandes

Créer une nouvelle demande
  1. Accédez au menu Demandes > Nouvelle demande
  2. Remplissez les informations requises :
    • Objet de la demande
    • Description détaillée
    • Organisme concerné
    • Type d'information demandée
  3. Joignez les documents justificatifs si nécessaire
  4. Cliquez sur Soumettre

Alertes Citoyennes

Signaler un problème

Le système d'alertes permet aux citoyens de signaler des problèmes de gouvernance ou de transparence.

Types d'alertes :
  • Corruption : Signalement d'actes de corruption
  • Mauvaise gestion : Problèmes de gestion publique
  • Non-transparence : Manque de transparence dans l'administration
  • Autre : Autres problèmes de gouvernance
Confidentialité : Toutes les alertes sont traitées de manière confidentielle. L'anonymat est garanti si souhaité.

Paramètres

Configuration utilisateur
Paramètres personnels :
  • Profil : Informations personnelles et contact
  • Notifications : Préférences de notification par email/SMS
  • Langue : Choix de la langue d'interface
  • Sécurité : Changement de mot de passe, authentification 2FA
Paramètres d'affichage :
  • Thème : Mode clair/sombre
  • Tableau de bord : Personnalisation des widgets
  • Listes : Nombre d'éléments par page
Conseil : Configurez vos notifications pour être informé en temps réel des mises à jour importantes.
