Questions Générales

SAMA CONAI est un module Odoo conçu pour promouvoir la transparence et la bonne gouvernance au Sénégal. Il permet aux citoyens de faire des demandes d'accès à l'information publique et de signaler des problèmes de gouvernance via un système d'alertes sécurisé.

SAMA CONAI est destiné à plusieurs types d'utilisateurs :
  • Citoyens : Pour faire des demandes d'information et signaler des problèmes
  • Agents publics : Pour traiter les demandes et gérer les réponses
  • Administrateurs : Pour configurer et maintenir le système
  • Organismes publics : Pour améliorer leur transparence

Oui, SAMA CONAI est un logiciel libre et open source. Il peut être utilisé gratuitement par toute organisation publique ou privée. Le code source est disponible sur GitHub sous licence AGPL-3.0.

Installation

Système requis :
  • Ubuntu 20.04+ / Debian 10+ / CentOS 8+
  • Python 3.8 ou supérieure
  • PostgreSQL 12+ (recommandé)
  • Minimum 4 GB RAM (8 GB recommandé)
  • Minimum 20 GB d'espace disque
  • Odoo 16 ou version supérieure

L'installation se fait en quelques étapes simples :
  1. Installez Odoo 16 ou supérieur
  2. Clonez le repository SAMA CONAI dans le dossier addons
  3. Exécutez le script d'installation : ./install.sh
  4. Redémarrez Odoo
  5. Activez le module dans l'interface d'administration
Consultez le Guide de Démarrage pour les détails complets.

En cas de problème d'installation :
  1. Vérifiez que tous les prérequis sont installés
  2. Consultez les logs d'Odoo pour identifier l'erreur
  3. Vérifiez les permissions sur les dossiers
  4. Assurez-vous que la base de données est accessible
  5. Contactez le support technique si le problème persiste

Utilisation

Pour créer un compte :
  1. Accédez à la page d'inscription sur le site
  2. Remplissez le formulaire avec vos informations personnelles
  3. Choisissez un mot de passe sécurisé
  4. Validez votre email via le lien de confirmation
  5. Votre compte sera activé après validation par un administrateur

Pour réinitialiser votre mot de passe :
  1. Cliquez sur "Mot de passe oublié" sur la page de connexion
  2. Saisissez votre adresse email
  3. Consultez votre boîte email pour le lien de réinitialisation
  4. Cliquez sur le lien et définissez un nouveau mot de passe
  5. Connectez-vous avec votre nouveau mot de passe

Oui, SAMA CONAI est disponible en plusieurs langues :
  • Français : Langue principale
  • Wolof : Langue nationale du Sénégal
  • Anglais : Pour les utilisateurs internationaux
Vous pouvez changer la langue dans vos paramètres utilisateur.

Demandes d'Information

Vous pouvez demander tout type d'information publique :
  • Documents budgétaires et financiers
  • Rapports d'activité des organismes publics
  • Données statistiques publiques
  • Procédures et réglementations
  • Informations sur les marchés publics
  • Données environnementales
Exceptions : Informations classifiées, données personnelles, secrets d'État.

Les délais de réponse varient selon le type de demande :
  • Informations simples : 15 jours ouvrables
  • Informations complexes : 30 jours ouvrables
  • Demandes nécessitant des consultations : 45 jours ouvrables
Vous recevrez une notification si un délai supplémentaire est nécessaire.

En cas de rejet de votre demande :
  1. Consultez la justification fournie par l'organisme
  2. Vous pouvez faire appel dans les 30 jours
  3. Reformulez votre demande si nécessaire
  4. Contactez la Commission d'Accès aux Documents Administratifs (CADA)
  5. En dernier recours, saisissez la justice administrative

Alertes Citoyennes

Oui, le système permet les signalements anonymes. Vos données personnelles sont chiffrées et votre identité est protégée. Cependant, fournir vos coordonnées peut aider à mieux traiter votre signalement.

Vous pouvez signaler :
  • Corruption : Pots-de-vin, détournements, abus de pouvoir
  • Mauvaise gestion : Gaspillage, négligence, inefficacité
  • Non-transparence : Refus d'information, opacité
  • Discrimination : Traitement inéquitable
  • Autres : Tout problème de gouvernance

Après votre signalement :
  1. Votre alerte est enregistrée et un numéro de suivi vous est attribué
  2. L'alerte est évaluée par une équipe spécialisée
  3. Une enquête peut être ouverte si nécessaire
  4. Vous êtes informé de l'avancement (si vous n'êtes pas anonyme)
  5. Des mesures correctives sont prises si le problème est avéré

Sécurité et Confidentialité

Oui, SAMA CONAI utilise plusieurs mesures de sécurité :
  • Chiffrement SSL/TLS pour toutes les communications
  • Chiffrement des données sensibles en base
  • Authentification à deux facteurs disponible
  • Logs d'audit complets
  • Sauvegardes régulières et sécurisées
  • Conformité RGPD

L'accès aux informations est strictement contrôlé :
  • Vos demandes : Vous et les agents assignés uniquement
  • Alertes anonymes : Équipe d'enquête spécialisée uniquement
  • Données personnelles : Administrateurs autorisés uniquement
  • Statistiques : Données anonymisées pour les rapports publics

Conformément au RGPD, vous avez le droit :
  • D'accéder à vos données personnelles
  • De rectifier des informations incorrectes
  • De demander la suppression de vos données
  • De vous opposer au traitement
  • De demander la portabilité de vos données
Contactez l'administrateur pour exercer ces droits.
